# Post-processing

This folder contains the spatial post-processing scripts used to refine the annual raster classification maps and convert them into vector datasets for CAMAF-2019-2025.

## Purpose

The post-processing workflow performs the following steps:

1. Applies a coastline-based land mask to remove landward false positives.
2. Removes small isolated patches using class-specific connected-component filtering.
3. Fills small internal holes within aquaculture patches.
4. Applies light morphological closing to selected classes.
5. Converts refined raster classification maps into vector polygons.
6. Assigns class attributes to the exported vector files.

## Input data

The main inputs are:

- Annual raster classification maps generated by the TransUNet inference workflow.
- A manually prepared coastline or landward mask vector dataset.
- Raster metadata, including spatial resolution, coordinate reference system, and spatial extent.

The raster classification maps use the following class IDs:

| Pixel value | Description |
|---|---|
| 0 | Background |
| 1 | Northern Rafts |
| 2 | Cages |
| 3 | Southern Rafts |
| 255 | Nodata, when available |

## Land masking

A coastline-based spatial mask is used to remove landward misclassifications. The coastline vector is rasterized to match the resolution, extent, and coordinate reference system of each annual classification raster. Pixels located within the landward mask are assigned to the background class.

## Small-patch removal

Small isolated patches are removed using 8-neighbour connected-component analysis. The class-specific minimum mapping-unit thresholds are:

| Class | Threshold | Area at 10 m resolution |
|---|---:|---:|
| Northern Rafts | 30 pixels | 3,000 m² |
| Cages | 15 pixels | 1,500 m² |
| Southern Rafts | 8 pixels | 800 m² |

## Hole filling

Small internal holes are filled using class-specific thresholds:

| Class | Threshold | Area at 10 m resolution |
|---|---:|---:|
| Northern Rafts | 100 pixels | 10,000 m² |
| Cages | 50 pixels | 5,000 m² |
| Southern Rafts | 30 pixels | 3,000 m² |

A light 3 × 3 morphological closing operation is applied to Northern Rafts and Cages to improve local patch continuity. No closing operation is applied to Southern Rafts to avoid excessive expansion of fragmented coastal patches.

## Vectorization

After spatial refinement, the raster classification maps are converted to vector polygons. Only pixels belonging to the three aquaculture facility classes are exported. Background and nodata pixels are excluded.

The output attribute table contains:

| Field | Description |
|---|---|
| `gridcode` | Class code. 1 = Northern Rafts; 2 = Cages; 3 = Southern Rafts |
| `class_name` | Abbreviated class label: N-fra, CA, or S-fra |
| `year` | Mapping year |

## Output

The outputs are annual ESRI Shapefile subsets. Each subset includes the standard `.shp`, `.shx`, `.dbf`, `.prj`, and `.cpg` components.

## Notes

Users should modify the input raster path, coastline vector path, output folder, and mapping year before running the scripts. The class-specific filtering and hole-filling thresholds were empirically determined through visual inspection of representative coastal regions and were kept constant for all annual maps.
